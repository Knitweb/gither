gither

The code forge authority for routing, review gates, PR enrichment, p2p manifests, and public documentation mirrors.

Start

python -m pip install -e .
gither repo-snapshot --repo .
gither gate --repo . --python-root src --strict

Concepts

  • GitHub and GitLab are mirrors; Gither owns review and release state.
  • Change notes preserve why a change exists before it becomes a PR.
  • Enrichment score ranks merged, reviewed contribution evidence without rewarding raw volume.

Guides

  1. Change flow Route, snapshot, edit, write a change note, test, and gate.
  2. PR enrichment Export merged PR evidence and run gither enrichment-score.
  3. Docs release Mirror docs/ to GitHub Pages and www.5mart.ml/gither.

Evidence

Accepted Gither PRs should include test output, docs links, review packs when useful, and enrichment export examples when contribution scoring changes.

Recovery
